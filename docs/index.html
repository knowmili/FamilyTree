Welcome to FamilyTree App

An application for generating and visualizing family trees.

Features

  • Add and delete family members
  • Visualize family tree
  • Save tree visualization as PNG

Installation

Dependencies

  • PyQt5

How to Run

  1. Clone the repository:
    2. git clone https://github.com/your-username/FamilyTreeGenerator.git
  2. Install dependencies:
    3. pip install PyQt5
  3. Run the application:
    4. python gui/tree.py
